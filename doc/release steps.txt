What steps to perform when releasing:

1)  Release via NuGet
2)  Create download on catelproject.com (with own download category)
3)  Create release page under "Download" section
4)  Release via Codeplex => just a forwarding release
5)  Release version in jira
6)  Update docs on confluence (create copy of LATEST space)
7)  Write blog post on catelproject.com
8)  Share via Twitter
9)  Share via LinkedIn
10) Update the following version numbers:
    * Version numbers in final builder
    * History.txt
    * SolutionAssemblyInfo.cs
